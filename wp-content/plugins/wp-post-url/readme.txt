This plugin is useful to change the url of main website, pages and posts as well as it also replace all OLD URL with NEW URL in whole database.

== Description ==

Sometimes you want to shift your domain or want to change your website URL like you have developed a website on demo url but now want to change that demo url with live URL. In these cases this plugin is really useful. 

This plugin replaces all found OLD URLs in database with New URL. So all images, files also come from the new URL. You will no need to check the whole website again that any URL in the database is OLD. So this plugin helps you to move a website or change old url with new in simple way.


== Installation ==

1. Upload 'wp-post-url' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the WP Post Url 
4. There will 2 textfield with Old URL and New URL label.
5. Please OLD URL which you want to replace and then New website URL in New URL field.
6. Click on "Submit" button.
7. Plugin will replace all OLD URL in database with New URL.
